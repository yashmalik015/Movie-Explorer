# Movie Explorer 🎬

## 📌 Description
Movie Explorer is a web application that allows users to search for movies and view their details using a public API.

## 🌐 API Used
OMDb API - https://www.omdbapi.com/

## ✨ Features
- Search movies by name
- Display movie title, poster, and year
- Responsive design

## 🔜 Upcoming Features
- Sorting (A-Z, Z-A)
- Filtering by year/type
- Dark mode
- Favorites using local storage

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- Fetch API

## 📁 Project Structure
movie-explorer/
│── index.html
│── style.css
│── script.js

## 🚀 How to Run
1. Open index.html in browser
2. Enter movie name
3. Click search
4. View results
